# HealthCenter_Chatbot
 # how to run
 ### Steps:

 clone the repository
 ```bash
 git clone https://github.com/Bhawna-04-dob/HealthCenter_Chatbot.git

 ### step 1: created a conda environment after opening repository in terminal
 ```bash
conda crete -n medibot python=3.9 -y
 ```

 ### step 2: activate the conda environment
 ```bash
 conda activate medibot
 ```

 ### step 3: install the requirements
 ```bash
 pip install -r requirements.txt
 ```

 ### step 4: run the app
 ```bash
 python app.py
 ```
